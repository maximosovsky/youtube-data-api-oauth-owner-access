# youtube-data-api-oauth-owner-access

Hermes skill for connecting owner-authorized YouTube channels through official YouTube Data API OAuth without exposing secrets in chat.

## Contents

- `SKILL.md` — reusable Hermes skill.
- `.gitignore` — blocks local OAuth secrets/tokens and downloaded Google client files.

## Safety

Do not commit OAuth token files, Google client-secret JSON files, cookies, passwords, or authorization codes.
